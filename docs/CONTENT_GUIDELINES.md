## 🔍 Content Audit Checklist (cho human review)

Mỗi batch content mới (AI gen hoặc manual) phải pass:

### Pre-publish checklist
- [ ] Đọc qua toàn bộ batch
- [ ] Search keyword cấm (Đài Loan, Tây Tạng, Hong Kong, Mao, etc.) → return 0
- [ ] Nếu có map → verify KHÔNG có lưỡi bò
- [ ] Nếu có ảnh historical → verify pre-1912
- [ ] Nếu nói về Trung Quốc → neutral, descriptive only
- [ ] Random sample 10% read aloud → có "kỳ" không?

### Auto-scanner script (đề xuất build)

```javascript
// js/utils/content-filter.js
const POLITICAL_KEYWORDS = [
  // Sensitive locations
  '台湾', '台獨', '西藏', '新疆', '维吾尔', '香港',
  '南海', '九段线', '黄岩岛', '钓鱼岛',
  // Historical events
  '六四', '天安门', '文化大革命', '大跃进', '法轮功',
  // Politics
  '中国共产党', '毛泽东', '习近平', '邓小平', '人民解放军',
  // Vietnamese sensitive
  'Hoàng Sa', 'Trường Sa', 'lưỡi bò', 'biển Đông',
  // English transliterations
  'Tibet', 'Xinjiang', 'Uyghur', 'Falun Gong', 'Tiananmen',
];

function scanContent(text) {
  const found = POLITICAL_KEYWORDS.filter(kw => 
    text.toLowerCase().includes(kw.toLowerCase())
  );
  if (found.length > 0) {
    throw new Error(`POLITICAL CONTENT DETECTED: ${found.join(', ')}`);
  }
  return true;
}
```

Chạy script này CI/CD trước mỗi deploy.

## ⚠️ Emergency procedure

- Action (remove/keep + explanation) trong 48h
